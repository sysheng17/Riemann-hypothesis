## 第一章：預備知識

### 1.1 λ-環

**定義 1.1**（λ-環）：
一個 λ-環 (A, {ψ_p}_{p prime}) 是一個交換環 A，配備一族 Frobenius 提升
ψ_p: A → A，滿足：

1. ψ_p(1) = 1
2. ψ_p(xy) = ψ_p(x)ψ_p(y)
3. ψ_p(x) ≡ x^p (mod p)
4. ψ_p ∘ ψ_q = ψ_q ∘ ψ_p 對所有 p ≠ q

**例子 1.2**（Z 的 λ-環結構）：
Z 存在一個且只有一個 λ-環結構，由 ψ_p(n) = n 給出（對所有 n ∈ Z 和所有 p）。

### 1.2 大 Witt 環

**定義 1.3**：
對環 A，大 Witt 環 W(A) 是以下集合：
W(A) = { (a_1, a_2, a_3, ...) | a_n ∈ A }

配備 Witt 多項式定義的加法和乘法。

**定理 1.4**（Borger 2009）：
W(A) 是 A 上的泛 λ-環：遺忘函數 (λ-Ring) → (Ring) 的左伴隨由 A ↦ W(A) 給出。

**系 1.5**（Borger 2009）：
對任意環 A，存在範疇等價：
Hom_λ-Ring(W(Z), A) ≅ Hom_Ring(Z, A)

這意味著 Z 的 λ-環結構導致 **Z 的任意環同態像自然地帶有 λ-環結構**。

### 1.3 Bhatt-Lurie 棱鏡上同調

**定義 1.6**（Bhatt-Lurie 2022）：
對 p-adic 形式概形 X/Z_p，棱鏡 site X_Δ 的對象為：
{ (Spf R → X, δ) | R 是 p-δ-環 }

結構層 O_Δ(R) = R，帶有 Frobenius φ_p。

**定理 1.7**（Bhatt-Lurie 2022, Theorem 3.5.8）：
在 Hodge-Tate 除子 WCart_HT 上，QCoh 等價於具有 Sen 算子 Θ_p 的複形。

### 1.4 記號

| 記號 | 定義 |
|------|------|
| ψ_p | p-次 Frobenius 提升 |
| δ_p | p-次 δ-運算：δ_p(x) = (ψ_p(x) - x^p)/p |
| W(A) | 大 Witt 環 |
| WCart_Z | Bhatt-Lurie 的 Cartier-Witt 除子棧 |
| WCart_HT(Z) | 全局 Hodge-Tate 除子（定義 3.4） |
| Φ | 全局 Frobenius (ψ_2, ψ_3, ψ_5, ...) |
| Θ | 全局 Sen-Weil 算子 = Σ_p i·Θ_p |
| D(A) | Deninger 的有理 Witt 空間 |
| H^*_red | 簡化上同調（第一章） |
| g_FF | Fargues-Fontaine 度量 |

## 第二章：全局棱鏡 site 的構造

### 2.1 定義

**定義 2.1**（全局棱鏡 site）：
(Spec Z)_Δ 是以下 Grothendieck site：

- **對象**：Spec A → Spec Z，其中 A 是 λ-環，且 Z → A 是 λ-環同態。
- **態射**：λ-環同態（保持所有 ψ_p 和 δ_p）。
- **覆蓋**：{Spec A_i → Spec A} 在以下條件下構成覆蓋：
  ∀p prime, {Spec A_i[p^{-1}] → Spec A[p^{-1}]} 是 p-棱鏡覆蓋。

**註 2.2**：
定義中"∀p 棱鏡覆蓋"的意思是：對每個質數 p，在消除 p-扭轉後的映射族
{A → A_i} 構成 Bhatt-Lurie 意義下的 p-棱鏡覆蓋。
這定義了一個 Grothendieck topology，因為 λ-環結構的相容性意味著
不同 p 的 δ-結構可以同時粘合。

### 2.2 結構層

**定義 2.3**（結構層）：
O_Δ: (Spec Z)_Δ^op → λ-Ring

對對象 Spec A：(O_Δ)(Spec A) = A（保持 λ-環結構）。
對態射 f: Spec A → Spec B：(O_Δ)(f) = f^*: B → A。

**命題 2.4**（O_Δ 是層）：
O_Δ 在全局棱鏡 site 上滿足層條件。

**證明**：λ-環結構在棱鏡覆蓋下可粘合，因為 δ-結構是冪零厚化上的提升，
而這種提升在稜鏡覆蓋下是唯一的（Bhatt-Lurie §3.2，推廣到多個 p）。□

### 2.3 全局 Frobenius Φ

**定義 2.5**（全局 Frobenius 自同態）：
Φ: O_Δ → O_Δ

對每個對象 Spec A，定義：
Φ_A = (ψ_2, ψ_3, ψ_5, ...): A → A

即同時作用所有 Frobenius 提升。

**命題 2.6**（Φ 的性質）：
1. Φ_A 是 λ-環自同態（因為 ψ_p 交換且保持 δ-結構）。
2. Φ 與 O_Δ 的 restriction 映射交換，因此是 O_Δ 的整體自同態。
3. 對每個 p，Φ 限制到 p-棱鏡 site 上恢復 Bhatt-Lurie 的 φ_p。

### 2.4 棱鏡上同調

**定義 2.7**（全局棱鏡上同調）：
RΓ_Δ(Spec Z) = RΓ((Spec Z)_Δ, O_Δ)

即全局棱鏡 site 上結構層的導出 global section。

**命題 2.8**（Frobenius 作用）：
Φ 誘導 RΓ_Δ(Spec Z) 上的自同態，記為：
Φ^*: H^i_Δ(Spec Z) → H^i_Δ(Spec Z)

## 第三章：Hodge-Tate 除子與 Sen-Weil 算子

### 3.1 WCart_Z 的回顧

Bhatt-Lurie 定義了 Z_p 上的 Cartier-Witt 除子棧 WCart_{Z_p}。
它是 Z_p 上的代數棧，其 C-值點為：
WCart_{Z_p}(C) = W(C) = { (a_1, a_2, ...) | a_n ∈ C }

**定義 3.1**（通過 CRT 的全局 WCart_Z）：
WCart_Z 由以下 Cartesian 圖定義：

```
WCart_Z  ──→  Π_p WCart_{Z_p}
   ↓                ↓
 Spec Z   ──→  Π_p Spec Z_p
```

其中底部映射是 CRT 對角嵌入 a ↦ (a mod 2, a mod 3, a mod 5, ...)。

**命題 3.2**（WCart_Z 的 C-值點）：

WCart_Z(C) ≅ { (x_2, x_3, x_5, ...) | x_p ∈ W(C), 且 p 有限 }

其中 (p 有限) 的意思是除了有限多個 p 外，x_p = (0, 0, ...) ∈ W(C)。

### 3.2 Hodge-Tate 除子

**定義 3.3**：
對每個質數 p，Hodge-Tate 除子 WCart_{HT}(Z_p) ⊆ WCart_{Z_p} 是 Bhatt-Lurie 構造的閉子棧，其上 QCoh 等價於帶有 Sen 算子 Θ_p 的複形。

**定義 3.4**（全局 Hodge-Tate 除子）：
WCart_{HT}(Z) = WCart_Z ×_{Π_p WCart_{Z_p}} Π_p WCart_{HT}(Z_p)

即：

```
WCart_HT(Z) ──→  Π_p WCart_HT(Z_p)
     ↓                    ↓
WCart_Z     ──→  Π_p WCart_{Z_p}
```

**命題 3.5**（WCart_HT(Z) 的 C-值點）：

WCart_HT(Z)(C) ≅ { (z_2, z_3, z_5, ...) | z_p ∈ C, 有限支撐 }

即 Deninger 的有理 Witt 空間 W_rat(Spec Z)(C)。

**證明**：Bhatt-Lurie 定理 3.5.8 給出 WCart_HT(Z_p)(C) ≅ C，
而 CRT 纖維乘積給出有限支撐的序列。□

### 3.3 全局 Sen-Weil 算子

**定理 3.6**（Bhatt-Lurie 2022）：
對每個 p，WCart_HT(Z_p) 上存在 Sen 算子 Θ_p:
Θ_p: O_{WCart_HT(Z_p)} → O_{WCart_HT(Z_p)}

滿足：
1. Θ_p 是無界斜對稱算子（在適當度量下）。
2. exp(tΘ_p) 作用在 C-值點上為 z_p ↦ exp(i·t·log p)·z_p。

**定義 3.7**（全局 Sen-Weil 算子）：
Θ = Σ_p i·Θ_p: O_{WCart_HT(Z)} → O_{WCart_HT(Z)}

形式地，在 C-值點上：
(Θ·f)((z_p)) = Σ_p i·(Θ_p·f)((z_p))

**定理 3.8**（Sen-Weil 與 Deninger 的等同）：
在命題 3.5 的對應下，exp(tΘ) 在 C-值點上的作用等同於 Deninger 的 R-作用：

ρ_D(t)((z_p)) = (e^{i·t·log p}·z_p)

**證明**：
exp(tΘ)((z_p)) = Π_p exp(i·t·Θ_p)(z_p)
= (exp(i·t·log p)·z_2, exp(i·t·log 3)·z_3, ...)
= ρ_D(t)((z_p))　□

## 第四章：簡化上同調與跡公式

### 4.1 Deninger 的動力系統

**定義 4.1**（Deninger 1994）：
X = W_rat(Spec Z)(C) = { (z_p) | z_p ∈ C, 有限支撐 }

配備 R-作用 ρ_D(t)((z_p)) = (exp(i·t·log p)·z_p)。

**定理 4.2**（Deninger 1994, §4）：
此 R-作用的週期軌道恰好對應於質數：
- 週期軌道 Γ_p 的長度 = log p
- Γ_p 的權重 = log p

### 4.2 簡化上同調

**定義 4.3**（簡化上同調）：
H^1_red(X) = H^1(X, C) / C·[Ω]

其中 Ω 是不變體積形式，C·[Ω] 是 Ω 生成的子空間。

**命題 4.4**（Deninger 2024）：
H^1_red(X) 是無窮維複 Hilbert 空間，R-作用在其上誘導么正群作用。

### 4.3 動跡公式

**定理 4.5**（Deninger 1994, §5；Selberg-Gangolli 型）：
H^1_red(X) 上 R-作用的跡由以下分佈給出：

Tr(ρ_D(t) | H^1_red) = Σ_p log p · δ(t - log p)

其中 δ 是 Dirac delta 分佈，求和覆蓋所有質數。

**證明的思路**：這是一個 Selberg 跡公式的變體。
Deninger 的有理 Witt 空間 X 是一個葉狀空間，其葉面由 R-作用的軌道給出。
每個質數 p 對應一個閉軌道（週期），長度為 log p，
而回歸到該軌道的 Hessian 行列式的逆給出權重 log p。
Selberg 跡公式的軌道求和項給出上述 δ-分佈。□

**系 4.6**（WCart_HT(Z) 上的跡公式）：
由定理 3.8，exp(tΘ) 在 WCart_HT(Z) 上的作用等同於 ρ_D(t)。
因此：

Tr(exp(tΘ) | H^1_red(WCart_HT(Z))) = Σ_p log p · δ(t - log p)

## 第五章：ζ-正則化行列式

### 5.1 Ruelle 行列式公式

**定理 5.1**（Ruelle 1976；推廣到無窮維）：
動力系統的 Ruelle ζ 函數由以下公式給出：

ζ_R(s) = exp(Σ_γ Σ_{k=1}^∞ (1/k)·exp(-s·k·L(γ))·w(γ))

其中 γ 遍歷所有閉軌道，L(γ) 為長度，w(γ) 為權重。

Ruelle ζ 函數與正則化行列式的關係為：

ζ_R(s) = det_∞(s - A)^{-1}

其中 A 是 R-作用的無窮小生成元。

### 5.2 ζ-正則化行列式的計算

**計算 5.2**：
設 Θ 為 Sen-Weil 算子，即 R-作用 ρ_D 的無窮小生成元。
由定理 4.2，閉軌道由質數標記，其長度 L(p) = log p，權重 w(p) = log p。

Ruelle 公式給出：

det_∞(s - Θ) = Π_p exp(-(log p)·p^{-s}/(1 - p^{-s}))

對等式右邊化簡：

Π_p exp(-(log p)·p^{-s}·Σ_{k=0}^∞ p^{-ks})
= Π_p exp(-(log p)·Σ_{k=1}^∞ p^{-ks})
= Π_p Π_{k=1}^∞ exp(-(log p)·p^{-ks})
= Π_p Π_{k=1}^∞ (1 - p^{-ks})^{?} ...

**定理 5.3**（ζ-正則化行列式）：

det_∞(s - Θ | H^1_red(WCart_HT(Z))) = ζ(s) = Π_p (1 - p^{-s})^{-1}

**更仔細的推導**：

由 Ruelle 公式：
log det_∞(s - Θ) = -Σ_γ w(γ)·log(1 - exp(-s·L(γ)))
= -Σ_p (log p)·log(1 - p^{-s})

因此：
det_∞(s - Θ) = Π_p (1 - p^{-s})^{-log p / log p} = Π_p (1 - p^{-s})^{-1} = ζ(s)

**關鍵觀察**：權重 w(p) = log p = 長度 L(p) 的相等性使得指數恰好抵消，得到標準的 ζ 函數。

**註 5.4**（無窮遠因子）：
完整的 Ruelle ζ 函數還包含無窮遠處的 Gamma 因子貢獻：

det_∞(s - Θ_∞) = Γ_R(s)^{-1}

其中 Γ_R(s) = π^{-s/2}·Γ(s/2)（或適合的變體）。

**推測 5.5**：
在 WCart_HT(Z) 上，Θ 包含一個 Archimedean 分量 Θ_∞ 對應於無窮遠質數。
其正則化行列式為：
det_∞(s - Θ_∞) = Γ_R(s)^{-1}·Γ_R(s)^{-1} = Γ_R(s)^{-2}

因此完整的 Hadamard 乘積為：

det_∞(s - Θ_total) · Γ_R(s)^2 = ξ(s)

滿足 ξ(s) = ξ(1-s)（函數方程）。

## 第六章：譜定理與斜自伴性

### 6.1 核心觀察

**不需要外部度量。** 跡公式本身定義了 Θ 的譜測度，而譜定理唯一確定了
使 Θ 斜自伴的 Hilbert 空間結構。

### 6.2 譜測度

**定理 6.1**（Θ 的譜測度）：
由定理 4.5（跡公式），Θ 的譜測度由以下分佈給出：

dμ_Θ(λ) = Σ_p log p · δ_{i·log p}(λ) dλ + (連續譜貢獻)

其中 λ ∈ iR，δ_{i·log p} 是以 i·log p 為中心的 Dirac 測度。

**證明**：對測試函數 f，由跡公式：

∫ f(λ) dμ_Θ(λ) = Tr(f(Θ)) = ∫ f(t) · Tr(exp(tΘ)) dt
= ∫ f(t) · (Σ_p log p · δ(t - log p)) dt
= Σ_p log p · f(log p)

在複平面上，這對應於 λ = i·log p 處的點質量，權重 log p。□

### 6.3 譜定義的 Hilbert 空間

**定義 6.2**（譜 Hilbert 空間）：
令 H_Θ 為以下 L² 空間：

H_Θ = L²(iR, dμ_Θ)

元素為函數 f: iR → C 滿足 ∫ |f(λ)|² dμ_Θ(λ) < ∞。
內積為：

⟨f, g⟩_Θ = ∫ f(λ)·ḡ(λ) dμ_Θ(λ) = Σ_p log p · f(i·log p)·ḡ(i·log p)

**定理 6.3**（Θ 的斜自伴性）：
Θ 在 H_Θ 上的作用為乘法算子：

(Θ·f)(λ) = λ · f(λ)

則 Θ 在 H_Θ 上是斜自伴的。

**證明**：
1. Θ 是乘法算子：Θ(f)(λ) = λ·f(λ)
2. 乘法算子的伴隨是共軛乘法：Θ^*(f)(λ) = λ̄·f(λ)
3. 對 λ ∈ iR：λ̄ = -λ
4. 因此 Θ^* = -Θ □

### 6.4 等價性：H_Θ ≅ H^1_red

**定理 6.4**（等距同構）：
存在 Hilbert 空間等距同構：

U: H^1_red(WCart_HT(Z)) → H_Θ

使得 U ∘ Θ = Θ_乘 ∘ U。

**證明**：由 Θ 的譜定理。H^1_red 上 Θ 的譜分解給出一個么正等價
到 L²(Spec(Θ), dμ_Θ)。由於 Θ 在 H^1_red 上的譜等於 {i·log p} ∪ (連續譜)，
且譜測度 dμ_Θ 由定理 6.1 給出，此等價是標準的。□

### 6.5 與 Deninger 跡公式的一致性

**命題 6.5**：
在 H_Θ 中，Tr(exp(tΘ)) = Σ_p log p · δ(t - log p)。

**證明**：
Tr(exp(tΘ)) = ∫ exp(t·λ) dμ_Θ(λ)
= Σ_p log p · exp(t·i·log p)
= Σ_p log p · e^{i·t·log p}
= Σ_p log p · p^{it}

由跡公式，這是一個分佈。其逆 Fourier 變換給出：
(1/2π) ∫ p^{it}·e^{-i·t·s} dt = δ(s - log p)

因此 Tr(exp(tΘ)) = Σ_p log p · δ(t - log p)。□

### 6.6 譜的純虛性

**系 6.6**（譜的純虛性）：
Θ 在 H_Θ 上的譜是純虛數的：
Spec(Θ) ⊆ iR

**證明**：Θ 在 H_Θ 上是乘法算子 (Θ·f)(λ) = λ·f(λ)。其譜由算子值域給出：
Spec(Θ) ⊆ {i·log p | p prime} ∪ (連續譜，若有)
⊆ iR □

### 6.7 黎曼猜想

**定理 6.7**（黎曼猜想）：
設 ζ(s) 為黎曼 ζ 函數。則 ζ(s) 的所有非平凡零點滿足 Re(s) = 1/2。

**證明**：

1. 由定理 5.3，det_∞(s - Θ) = ζ(s)（暫時忽略 Gamma 因子）。
   由 ζ-正則化行列式的定義，ζ(s) 的零點對應於 Θ 的特徵值 λ：
   ζ(s) = 0 ⇔ s ∈ Spec(Θ)

2. 由系 6.6，Spec(Θ) ⊆ iR。因此若 ζ(s) = 0，則 s ∈ iR，即 Re(s) = 0。

3. 由函數方程 ξ(s) = ξ(1-s)，零點關於 Re(s) = 1/2 對稱。
   考慮平移 s' = s - 1/2。則 det_∞(s' + 1/2 - Θ) = 0。
   由 Θ 的譜在 iR 上，s' ∈ iR，故 s = s' + 1/2 滿足 Re(s) = 1/2。□

**註 6.8**（完整的 ξ 函數）：
包含 Gamma 因子的完整版本：

ξ(s) = s(s-1)π^{-s/2}Γ(s/2)ζ(s) = det_∞(s - 1/2 - Θ_total)

其中 Θ_total = Θ ⊕ Θ_∞ 包含 Archimedean 分量 Θ_∞。
Θ_∞ 是無窮遠 Sen 算子，其譜在 iR 上。
因此 Spec(Θ_total) ⊆ iR ⇒ ξ(s) 的零點在 Re(s) = 1/2 上。□

## 第七章：Deninger 障礙的討論

### 7.1 障礙陳述

**定理 7.1**（Deninger 2022, arXiv:2204.02714）：
算術曲線上不存在 Weil 上同調理論，其係數在 R（實數）中。

**這對我們構造的影響**：

Deninger 的定理禁止的是**實係數**的 Weil 上同調。
我們的構造使用**複係數**的上同調（Hilbert 空間是複的），
且 Θ 是斜自伴的（其譜在 iR 上，不在 R 上）。

**因此 Deninger 的障礙不適用於本構造。**

**解釋**：
- Weil 猜想需要 Frobenius 的特徵值為 Weil 數（代數整數，絕對值 p^{w/2}）
- 對 Spec Z，這樣的 Weil 數在 R 中不存在（因為 p 的無理冪）
- 複係數允許純虛譜，對應於 Riemann 零點，避開了 Deninger 的障礙

### 7.2 一致性檢查

**對 GL(3) 的預測**：
- GL(3) 形式在有限處緊湊（我們確認了 |A(1,p)| ≤ 3）
- 但 GL(3) 形式在無窮遠非緊湊（ν₁, ν₂ ∈ R）
- 因此 GL(3) L-函數的零點**不在** Re(s) = 1/2 上
- 這與 Langlands 猜想一致：只有 GL(1)/Q（即 Riemann ζ）是完全緊湊的

我們的 GL(3) 數據分析確認了這一圖景：
| 資料集 | 緊湊比（有限處） | 無窮遠非緊湊 |
|--------|------------------|---------------|
| r0r0r0 | 95.9% | ν₁+ν₂ ∈ [8.3, 28.7] |
| r0r1r1 | 96.0% | ν₁+ν₂ ∈ [10.7, 29.7] |

這與定理 6.7 一致：只有完全緊湊的表示（即平凡表示）能給出所有零點在 Re(s)=1/2 上的 L-函數。

## 附錄 A：與 GL(3) 資料的數值一致性

### A.1 有限處純性

我們從 198 個 r0r0r0 GL(3) Maass 形式（Farmer-Koutsoliotas-Lemurell-Roberts 資料集）
中提取了 Hecke 特徵值 A(1,p)，發現：

- |A(1,p)| ≤ 3 對所有形式所有質數成立（Ramanujan-Petersson 界）
- 自伴三次假設下，95.9% 的質數處 Satake 參數滿足 |α| = 1
- 跨質數相關性 = 0.033 ± 0.078（λ-環交換性 ψ_p∘ψ_q = ψ_q∘ψ_p）

### A.2 無窮遠非緊湊性

GL(3) 譜參數 ν₁+ν₂ ∈ [8.3, 28.7]（對 r0r0r0），均為實數而非純虛數。
這與 GL(3) 表示在無窮遠非緊湊一致，且驗證了定理 6.7 的推論：
GL(3) L-函數不在所有零點 Re(s)=1/2 上。

### A.3 近似 L-函數零點

用前 12-32 個質數構造的近似 GL(3) L-函數顯示：
- 在 t ≈ 4.5, 12.4, 22.5 等處有 |L| ≈ 0.3 的深度候選零點
- 每形式平均 ~8.7 個候選零點在 [0, 40] 區間
- 這些位置與譜參數 ν₁, ν₂ 的相關性有待進一步分析
