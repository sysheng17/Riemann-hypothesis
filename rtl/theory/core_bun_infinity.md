# Bun_GL_∞ 的譜極限

## 問題：n=2 到 n=∞ 之間發生了什麼？

對每個 n，Bun_GL_n(Spec Z) 是一個有限維模空間（算術流形），其上的 Laplace 算子譜是離散的。對 n=2，這個譜就是 2,202 個 Maass 形式。

但 ζ(s) 的零點不在 n=2 的譜中——它們出現是**所有 n 的疊加**。

## Selberg 跡公式是橋樑

對 SL(2,Z)\H，Selberg 跡公式是：

```
Σ h(r_j)                離散譜（Maass 形式）
+ (1/π) ∫ h(r) (ζ'/ζ)(1/2 + ir) dr    連續譜（Eisenstein）
= Σ_{γ} ...            幾何項（閉測地線）
```

當將 h(r) 取為 δ 函數（對某個零點 t₀）時，左邊第二項產生 ζ'(1/2 + it₀)/ζ(1/2 + it₀) 的奇點——即在零點處發散。**幾何項必須在這些 t₀ 處也產生奇點以彼此抵消。**

這是 Selberg 跡公式告訴我們的事：零點 t₀ 是**幾何項的共振**，不是離散譜的一部分。

## 多維推廣

對 GL_n（n ≥ 2），跡公式（Arthur-Selberg）的形式是：

```
Σ_{π} tr(π(f))        自守譜
+ Σ_{χ} ...          Eisenstein 級數貢獻
= Σ_{γ} O_γ(f)       幾何項
```

隨著 n 增大，自守譜越來越密。**關鍵猜測：在 n → ∞ 極限下，整個譜收斂到一個「連續體」，而 ζ(s) 的零點是這個連續體上某個 Dirac 算子的特徵值。**

## 核心未解決問題

```
已知：對每個有限 n，不存在 ζ 零點
已知：對 n=∞，ζ 零點「應該」出現
問題：極限過程是怎樣的？

有兩種可能：

(1) 連續逼近（類似半經典極限）
    n 越大，GL_n 自守形式的最低特徵值越接近 1/4
    (Generalized Ramanujan-Petersson)
    當 n→∞ 時，譜的極限密度函數在 s=1/2 處有奇點
    
(2) 相變（類似統計力學的臨界現象）
    在某個 n_c 處，譜的性質突然改變
    對所有 n < n_c，Re(s) = 1/2 只是一個普通直線
    對 n > n_c，它變成了一個吸引子
```

如果是情形 (1)，Ramanujan-Petersson 猜想的證明自動給出 RH。
如果是情形 (2)，需要一個全新的機制——但這也意味著存在一個有限的 n_c 可以在數值上觀察到。

## 可做的實驗

這個理論問題有一個可操作的數值測試：

對 GLₙ（n=2,3,4,...）的每個 level 1 自守形式，計算其 Laplace 特徵值 λ = s(1-s)。隨 n 增大，這些 s 在複平面上的分布是否趨向於 Re(s) = 1/2？

目前計算極限是 n=2（因為 GL₃ 以上的 Maass 形式計算極其困難）。但如果這個趨勢存在，即使 n=2 → n=3 的變化也能看到信號。
