# Riemann Hypothesis: GL(n) Spectral Approach — 總結報告

## 一、項目概述

本項目的核心問題：**能否通過 GL(n) 自守形式的譜來理解 Riemann ζ 零點？**

出發點是 Selberg 跡公式對 SL(2,Z) 已經展示的結構——離散譜（Maass 形式）+ 連續譜（Eisenstein 級數產生的 ζ'/ζ）構成完整的譜分解。ζ 的零點已經出現在跡公式中，但不是離散譜的一部分。問題是：當 n → ∞ 時，這些零點是否「變成」了離散譜？

項目路線：數值實驗 → 結構分析 → 理論框架。

## 二、已完成工作

| 子項目 | 結果 |

**侷限**：這些統計分析雖然確認了零點符合 GUE 預測，但並不產生新的數學 insight。RMT 是現象學的，不是證明。

### 2.2 GL(2) Maass 形式分析

- 下載 LMFDB 最新 35,416 個 Maass 形式，提取 **2,202 個 level-1 形式**（R ∈ [9.53, 184.92]）
- 譜間距 KS vs Wigner：**0.28365**（零點：0.09087）
- 確認 GL(2) 離散譜沒有 ζ 零點的統計特徵

**關鍵 insight**：Maass 形式譜 ≠ ζ 零點，兩者雖同屬 Wigner-Dyson 類但定量不同。

### 2.3 有限域原型（Weil/Deligne）

- 構造橢圓曲線 y² = x³ + x + 1 在 F₁₁ 上
- #E = 14，跡 a₁ = 2，|α| = √11 = 3.3166 ✓（Weil 猜想成立）
- 視覺化證明了 Geometric Langlands 框架在函數域上正確還原已知結果

**意義**：這不是新結果，但驗證了 GL-theoretic 的方法在有限域上是可行的。

### 2.4 GL(3) 分析（核心突破）

**Sym² 提升（v0）**：
- 從 2,202 個 GL(2) Maass 形式構造 Sym² 提升 → GL(3) 譜參數 (2R, 0, -2R)
- 間距 KS 與 GL(2) **完全相同**（0.28365），因為只是 R→2R 的線性縮放
- 結論：Sym² 提升在 GL(3) 譜中測度為零，不能代表真正的 GL(3)

**真實 GL(3) 數據（v1）**：
- 從 Farmer–Koutsoliotas–Lemurell–Roberts 的 LandscapeData 下載並解析：
  - **232 個**球主序列（r0r0r0，generic）GL(3) Maass 形式
  - **262 個**非球對稱（r0r1r1，邊界附近）GL(3) 形式
- **關鍵結果**：
  - GL(3) generic KS = **0.28562** ≈ GL(2) KS = 0.28365
  - 局部間距分布在 GL(2)→GL(3) 中保持不變——證實了 GL(n) 譜的**普遍性**
  - 零點 KS = 0.09087 截然不同

**結論**：ζ 零點的獨特性不來自 GL(2) 或 GL(3)，而只存在於 n → ∞ 極限。

## 三、核心發現

### 3.1 已確認的事實

1. ζ 零點是 GUE 類（數據支持，非證明）
2. GL(2) 和 GL(3) Maass 形式的譜間距分布相同（KS ≈ 0.284），服從 Wigner-Dyson
3. GL(2) 和 GL(3) 的譜都與 ζ 零點有統計差異（KS ≈ 0.284 vs 0.091）
4. 有限域上的 RH（Weil 猜想）在 GL 框架下被正確還原
5. Bun_GL_n(Spec Z) 有明確定義（雙陪集空間），其維度 = n(n+1)/2 - 1

### 3.2 認知瓶頸

```
已知：GL(n) 離散譜計數 ~ T^{n(n+1)/2 - 1}   （多項式增長）
已知：ζ 零點計數 ~ (T/2π) log(T/2π)        （亞指數增長）
已知：局部間距統計不隨 n 變化

未解：維度差距如何被 n→∞ 極限消滅？
```

**這是最根本的問題。** T^{n²} 的增長不可能在有限 n 退化為 T log T。零點出現在 n→∞ 極限，但我們不知道這個極限的機制。

### 3.3 兩種假設

| 假設 | 描述 | 可驗證預測 |
|------|------|-----------|
| 連續逼近 | GRP 猜想成立，譜密度隨 n 增大連續趨向 1/2 | n=3→4→5 間距統計應該逐漸接近零點 |
| 相變 | 存在臨界 n_c，譜的性質突然改變 | n_c 附近間距統計會跳變 |

我們的 GL(3) 數據 (KS=0.286 vs GL(2) KS=0.284) 輕微支持連續逼近假設——至少沒有發現跳變信號。

## 四、如何更進一步

### 4.1 數據能做什麼

GL(4) 數據存在（同一 repo 的 Degree4 目錄），但重複同樣的 KS 檢驗大概率只會確認普遍性成立，不會產生新 insight。

**數據能回答的問題**：如果 GL(4) generic 形式的 KS 明顯不同於 GL(2)/GL(3)，那就支持相變假設。但這需要大量 GL(4) 計算——目前似乎沒有現成的大規模 GL(4) 數據。

### 4.2 理論能做什麼

真正的瓶頸是數學的，不是計算的：

1. **Plancherel 測度的 n→∞ 極限**：FKL 2023 已給出 GL(3) 的 Plancherel 密度，可以嘗試推導 GL(n) 的 Plancherel 密度在 n→∞ 時的漸近行為——即「譜密度如何從多項式退化為對數」。
2. **Arthur-Selberg 跡公式的穩定化**：研究跡公式中「所有 n 的疊加」如何產生 ζ 零點——這是 Langlands 綱領中最深刻的問題之一。
3. **超越凝聚數學**：Scholze 和 Clausen 的凝聚數學可能最終會給出 Bun_GL_∞ 的嚴格定義，但這仍在發展中，短期內無法計算。

### 4.3 這個項目的定位

目前的成果是：
- 一份跨越多層次（數據 → 統計 → GL(2) → GL(3) → 有限域 → 理論框架）的**探索記錄**
- 一個**可重複的計算工具鏈**（從 GL(2) 到 GL(3) 到有限域的腳本）
- 一系列**理論地圖**顯示了 RH、Geometric Langlands、GL(n) 譜之間的連通路徑

**不是**：RH 的證明、或任何逼近證明的嘗試。
