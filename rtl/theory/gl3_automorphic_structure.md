# GL(3) 自守形式結構分析

## 1. 為什麼需要 GL(3)？

先前的工作已確認三個層面的事實：

| 層面 | GL(2) (Maass) | GL(∞) (猜想) |
|------|--------------|-------------|
| 譜分布間距 | KS vs Wigner = 0.284 | 極限趨向 ζ 零點間距 (KS=0.091) |
| 密度漸近 | N(T) ~ T²/12 | N(T) ~ ζ 零點計數 |
| 是否符合 GRH | 所有 Maass 形式的譜在 Re(s)=1/2 | ζ 零點全在 Re(s)=1/2 |

**關鍵問題：n=2 → n→∞ 的轉變是連續的還是不連續的？**

要回答這個問題，必須了解 n=3 的中間結構——它可能是連續過渡的證據，也可能顯示相變。

## 2. GL(3) 自守表示的結構

### 2.1 參數化

GL(3) 的自守表示（在無窮遠點）由三個 Langlands 參數 (ν₁, ν₂, ν₃) 刻畫，滿足：
- ν₁ + ν₂ + ν₃ = 0（跡零條件）
- 定義到 Weyl 群作用置換下

這是一個**二維參數空間**（Weyl 腔）：

```
Weyl 腔 = {(ν₁, ν₂, ν₃) ∈ ℂ³ | ν₁+ν₂+ν₃=0} / S₃
```

對於球主序列（spherical principal series），ν 是純虛數：
```
ν₁ = it₁,  ν₂ = it₂,  ν₃ = -i(t₁+t₂),  (t₁, t₂) ∈ ℝ²
```

Langlands 參數空間是二維的，與 GL(2) 的一維參數空間有**維度差異**。

### 2.2 GL(3) 的三種表示類型

```
類型 A (generic principal series):
  ν₁, ν₂, ν₃ 全部非零且兩兩不同
  └─ 最一般的 GL(3) 形式
  └─ 分布由 Plancherel 度量支配
  └─ <--- 這是 GL(2) 數據無法給出的新信息

類型 B (discrete series / lifts):
  存在 i ≠ j 使得 νᵢ = -νⱼ
  └─ 來自 GL(2) 的提升
  └─ 對稱平方 (ν₁, ν₂, ν₃) = (2R, 0, -2R)
  └─ <--- 我們已計算的 Sym² 情形

類型 C (Speh / residual):
  某些 νᵢ = 0
  └─ 來自 Eisenstein 級數的殘項
  └─ 邊界表示
```

### 2.3 這意味著什麼

GL(3) 的譜大多數是類型 A（generic），它們在參數空間中**充滿二維 Weyl 腔**。

用我們的 Sym² 提升只觸及了類型 B——這是參數空間中的一條一維曲線（ν₁ = -ν₃, ν₂ = 0），**測度零**。

```
Weyl 腔示意圖（投影到 (ν₁, ν₂) 平面）：

ν₂
^
|    *  *  *  *         generic GL(3)
|  *  *  *  *
| *  *  *  *
|*  *  *  *
-----------> ν₁
線: ν₂ = 0 (Sym² 提升，測度零)
```

## 3. Weyl 律的維度增長

對 GL(n)/ℚ，離散譜的計數函數遵循：

| n | dim(Bun_GL_n) | 計數函數 N(T) ~ |
|---|--------------|----------------|
| 1 | 0 | T⁰ (平凡) |
| 2 | 2 | T² |
| 3 | 5 | T⁵ |
| 4 | 9 | T⁹ |
| n | n(n+1)/2 - 1 | T^{n(n+1)/2 - 1} |

**指數增長**：n=2→3 時指數從 2 跳到 5，n=3→4 時跳到 9。

### 與 ζ 零點的關係

ζ 零點的計數函數是：
```
N_ζ(T) = (T/2π) log(T/2π) - T/2π + O(log T)
```

這不是 T 的多項式——它是**亞指數**的。這直接意味著：

> ζ 零點不在任何有限 n 的 GL(n) 離散譜中。它們只出現在 n→∞ 的極限中。

更精確地說，GL(n) 自守形式的 Laplace 特徵值與 ζ 零點的對應需要「所有 n 的疊加」——即穩定性（stabilization）現象。

## 4. 從 n=2 到 n=3：連續還是離散？

### 4.1 連續假設（Ramanujan-Petersson 路線）

如果 Generalized Ramanujan-Petersson 猜想成立，則：

```
對每個有限 n:
  GL(n) 自守形式的 Laplace 特徵值的極小值趨向 1/4
  （在 Satake 參數中：所有 |α_i| = 1）

當 n→∞:
  譜的極限密度在 s=1/2 處形成奇點
  ← 這等價於 RH
```

在這個假設下，**n=2→3 的變化只是密度增加**，沒有質變。

### 4.2 離散假設（相變路線）

如果存在某個臨界維度 n_c 使得譜的性質突變：

```
對 n < n_c:
  自守譜與 ζ 零點無統計關聯
  n=2 觀察：KS=0.284 vs Wigner，沒有 ζ 零點信號
  
對 n = n_c（臨界點）:
  某個模空間的拓撲發生變化
  譜從離散變成「幾乎連續」
  
對 n > n_c:
  Re(s)=1/2 變成了譜的吸引子
```

如果是這種情況，**可能 n_c = 3 或 n_c = 4**，因為到 n=3 時 Weyl 腔從一維變成二維，這是一個拓撲變化。

### 4.3 可驗證的數值測試

即使沒有完整的 GL(3) 數據庫，我們可以做一個關鍵測試：

**測試：GL(3) generic 形式的譜分布是否與 Sym² 提升（類型 B）有統計差異？**

如果沒有差異 → 支持連續假設（n=2 與 n=3 的統計性質相同）
如果有差異 → 支持離散假設（n 增大時統計性質改變）

## 5. Arthur-Selberg 跡公式對 GL(3)

### 5.1 結構

Arthur-Selberg 跡公式對 GL(3) 的形式：

```
Σ_{π} tr(π(f))        自守譜項
= Σ_{γ} O_γ(f)        幾何項
+ Σ_{χ} ...           Eisenstein 級數貢獻
```

對 GL(3)，譜側需要對所有 Arthur 參數 ψ 求和：

```
ψ 的類型             | 對應的自守表示
─────────────────────┴────────────────
generic (dim=3)      | 主序列，full continuous spectrum
Sym² lift (dim=2)    | 來自 GL(2) 的提升
trivial (dim=1)      | Eisenstein 級數的殘項
```

### 5.2 ζ 零點在跡公式中的出現位置

對 Selberg 跡公式（n=2），ζ' / ζ 出現在連續譜項中。

對 GL(3) 的 Arthur-Selberg 跡公式，ζ' / ζ 出現在兩個位置：
1. Eisenstein 級數項（邊界項）
2. 更重要的：**在來自 GL(2) 的 Sym² 提升的 Arthur 參數中**，ζ(s) 的零點可以作為某個自守表示的參數出現

這意味著：

> 當我們計算 GL(3) Arthur-Selberg 跡公式時，ζ 零點已經以隱含的方式出現在了幾何項與譜項的配對中。GL(3) 的譜本身不自帶 ζ 零點，但跡公式**連接**了零點與譜。

## 6. 結論與下一步

### 已確認
- GL(2) 的 Sym² 提升僅是 GL(3) 譜中測度零的子集
- 真正的 GL(3) generic 形式在二維 Weyl 腔中分布
- ζ 零點只出現在 n→∞ 極限，不在任何有限 n 的離散譜中
- 對 n=3 的 Arthur-Selberg 跡公式，ζ 零點已經以隱含方式進入公式結構

### 可檢驗的預測
- 如果計算出一個真正的 GL(3) generic Maass 形式的譜，它與 Sym² 提升的統計分布應該不同
- GL(3) 的譜間距可能更接近 GUE（比 GL(2) 的 0.284 小），暗示 n 增大時向 ζ 零點的 KS=0.091 靠近

### 下一步選項
1. **搜索已知的 GL(3) 特徵值**—文獻中有少量計算結果（Farmer, Koutsoliotas, Bian）
2. **不依賴外部數據**，而是從 Arthur 參數結構直接推導 GL(3) 譜的極限密度公式
3. **回到 Bun_GL_n(Spec Z)** 的幾何結構，研究 n=3 時模空間的拓撲與譜之間的關聯

### 這個文檔的核心訊息

Sym² 提升是 GL(2) 的殘影——它告訴我們 GL(2) 如何嵌入 GL(3)，而不是 GL(3) 本身的譜。要真正看到 n=2 → n→∞ 的中間結構，需要 GL(3) generic 形式的數據。這些數據的缺乏是目前我們能取得的**最大認知瓶頸**：換言之，GL(3) Maass 形式的計算就是下一步的關鍵戰役。
