# T1：λ-δ-環 + 葉狀稜鏡上同調

## 核心想法

W_rat(Spec Z)(C) 上有兩層代數結構：
- **λ-環結構**：Adams 運算 ψ_p 作用在 Witt 向量分量上
- **δ-環結構**：p-導子 δ_p(x) = (ψ_p(x) − x^p)/p

稜鏡上同調（Bhatt-Lurie）關於每個質數 p 給出 QCoh(WCart_HT(Z_p)) ≅ {帶有 Sen 算子 Θ_p 的複形}。
目標：用 λ-δ 結構構造**全局**稜鏡上同調，並給出 H¹ 上的度量。

---

## 1. 代數結構

### 1.1 Adams 運算與 R-作用交換

在 W_rat(C) 的冪級數表示下，f(t) ∈ 1 + tC[[t]]：
```
ψ_p(f)(t) = f(t^p)
φ^t(f)(t) = f(e^{it}·t)
```

**引理 1.1.1**：φ^t ∘ ψ_p = ψ_p ∘ φ^t。

*證明*：ψ_p(φ^t(f))(t) = φ^t(f)(t^p) = f(e^{it}·t^p) = f(e^{it}·t^p)
φ^t(ψ_p(f))(t) = ψ_p(f)(e^{it}·t) = f((e^{it}·t)^p) = f(e^{ipt}·t^p)
這兩個不相等！ψ_p 和 φ^t **不交換**——除非 p=1。

更正：應該用 ghost 分量 w_n ∈ W_rat(C)。Adams ψ_p 在 ghost 分量上：
ψ_p(w_n) = w_{np}

R-作用在 ghost 分量上：
φ^t(w_n) = e^{it·log n}·w_n

計算：
ψ_q∘φ^t(w_n) = ψ_q(e^{it·log n}·w_n) = e^{it·log n}·w_{nq}
φ^t∘ψ_q(w_n) = φ^t(w_{nq}) = e^{it·log(nq)}·w_{nq} = e^{it·(log n + log q)}·w_{nq}

**這也不等**——除非 e^{it·log q} = 1 對所有 t，即在 t=0。

所以 ψ_q 和 φ^t **不交換**。正確的關係是：
φ^t∘ψ_q = ψ_q∘φ^{t·(log q)/?}... 不，更簡單地：
φ^t∘ψ_q = e^{it·log q}·(ψ_q∘φ^t)

即：φ^t∘ψ_q = ψ_q∘φ^t·(eq 因子量綱變換)

**結論**：Adams 運算與 R-作用在 ghost 分量上以乘法特徵因子相關聯。

### 1.2 權分次的來源

λ-環的 Adams 運算 ψ_p 在 H¹ 上有特徵值，應等於 p^{ρ-½}（對應於 Θ 的特徵值 ρ）。
這是因為 Hecke 運算元與 Langlands 對應。

權分次：
```
H⁰: weight 0, ψ_p = 1
H¹: weight 1, ψ_p = p^{Θ-½} (as operators)
H²: weight 2, ψ_p = p·id
```

### 1.3 從 λ-環正性到 Hilbert 度量

λ-環的 **Schur 正性**：在表示論中，一類 λ-運算在 Schur 多項式下正。
這給出複表示環 R(G) 上的正錐。

對 W_rat(C)，λ-環結構來自 Witt 向量的普遍性質。
「λ-正性」猜測：存在某個正錐 P ⊂ W_rat(C) 使得：
- ψ_p(P) ⊂ P
- 對所有 x ∈ H¹，⟨x, x⟩ = 某種「λ-跡」> 0

**問題**：λ-環正性對無限維空間沒有自然的定義。
表示論的 Schur 正性是對有限維表示而言的。

---

## 2. 全局稜鏡構造

### 2.1 局部稜鏡上同調

對每個質數 p，稜鏡上同調 Hⁱ_Δ(Spec Z_p) 是 A_inf = W(C_p) 上的模。
Bhatt-Lurie 定理（定理 3.5.8）：
QCoh(WCart_HT(Z_p)) ≅ {具有 Sen 算子 Θ_p 的複形}

### 2.2 全局粘合問題

嘗試：
```
Hⁱ_Δ(Spec Z) := ∏̂'_p Hⁱ_Δ(Spec Z_p)  (限制乘積)
```

但這是一個純形式構造，沒有幾何內容。
沒有已知的方式將各質數的稜鏡上同調粘合成一個能承載 R-作用的全局對象。

### 2.3 葉狀稜鏡上同調

如果我們能定義 S_Z 上的「稜鏡化葉狀結構」A_Δ，其葉狀 de Rham 上同調是：
Hⁱ_F(S_Z, A_Δ) = Hⁱ_F(S_Z) ⊗ Hⁱ_Δ(Spec Z)  （某種 Künneth）

則乘積結構給出：
- 來自 S_Z 的 R-作用
- 來自 Hⁱ_Δ 的 Frob 作用
- 來自 λ-δ 環的正性

**但即使在光滑代數幾何中，稜鏡化葉狀上同調也不存在。**

---

## 3. 評估

| 子步驟 | 狀態 |
|--------|------|
| Adams 運算 ψ_p 與 φ^t 交換 | ❌ 不交換（相差特徵因子） |
| λ-環正性 → Hilbert 度量 | ❌ 無限維無定義 |
| 全局稜鏡上同調 Hⁱ_Δ(Spec Z) | ❌ 不存在 |
| 葉狀稜鏡上同調 | ❌ 不存在 |
| δ-環正性 | ❌ δ-環沒有正性概念 |

**結論**：T1 不可行。每個關鍵步驟要麼是錯誤的（ψ_p 與 φ^t 交換），要麼不存在（全局稜鏡上同調）。

# T3：顯式公式的加法分解

## 核心想法

Weil 顯式公式的右邊是質數貢獻的和：
Σ_p log p·δ(t − log p) + Γ-項

對每個質數 p，構造局部算子 Θ_p 使得 Tr(e^{tΘ_p}) = log p·δ(t − log p)。
則 Θ = Σ_p Θ_p 自動滿足全局跡公式。

---

## 1. 局部構造

### 1.1 逐質數 Hilbert 空間

對每個質數 p，設 H_p ≅ ℂ（1維），Θ_p 在 H_p 上的特徵值 = log p。
則：
Tr(e^{tΘ_p} | H_p) = e^{t·log p} = p^t

這不是 δ(t−log p)！問題：δ(t−log p) 是分佈，不是函數。

對 H_p = L²(R/log p Z)，Θ_p = −i·d/dx（在圓周上）。
Tr(e^{tΘ_p}) = Tr(e^{-it·d/dx}) = Σ_{n∈Z} e^{it·n/log p} = Σ_{n∈Z} δ(t − n·log p)

但 Σ_{n∈Z} δ(t − n·log p) ≠ log p·δ(t−log p)。
多了所有整倍數 n·log p 的 δ 函數！

### 1.2 修正：Poisson 求和

用 Poisson 求和公式：
Σ_{n∈Z} δ(t − n·log p) = (1/log p)·Σ_{k∈Z} e^{2πik·t/log p}

我們需要的只是 k=0 項：1/log p。
所以需要一個篩選運算元來選出 k=0 項。

這需要一個「投影」運算元 Π_p: L²(R/log p Z) → H_p（1維）：
Π_p(f) = (1/log p)·∫_0^{log p} f(t) dt

則 Tr(Π_p∘Θ_p∘Π_p) = log p·δ(t−log p)。

但 Π_p 是積分核為常數的運算元——它不是對角的。
全局的 Θ = Σ_p Π_p∘Θ_p∘Π_p 不自伴。

### 1.3 加法分解的譜

設 Θ = Σ_p Θ_p 在 H = ⊕ H_p 上（直和）。
則 Θ 的譜是 {log p : p prime} ∪ {0}。

這是一個離散集合——**不是 ζ 零點**。

---

## 2. 需要耦合

為了讓 Θ 的譜變成 ζ 零點，各質數分量需要**耦合**。
耦合對應於解析延續——正是 Ruelle 障礙中的問題。

### 2.1 耦合操作

設 H = ⊕_p H_p，定義耦合運算元 C: H → H：
(C(x))_p = Σ_{q≠p} c_{pq}·x_q

其中 c_{pq} 是耦合參數。全局運算元 Θ_total = Θ_diag + C。

對譜的影響：Θ_diag 的譜 {log p} 被 C 擾動後變成 {ρ}（ζ 零點）。

**需要**：找到 c_{pq} 使得 Θ_total 的譜 = {ρ : ζ(ρ)=0}。

但這就是在用零點定義 c_{pq}——循環定義。

### 2.2 非循環定義的嘗試

如果 c_{pq} 由數論函數給出，例如：
c_{pq} = gcd(p,q) 或 log(p+q) 或某種 L-函數的值

這些猜測可以數值測試：
- 計算耦合矩陣 C 對 Θ_diag 譜的擾動
- 看特徵值是否接近 ζ 零點

**問題**：沒有先驗理由選擇特定的耦合。

---

## 3. 評估

| 子步驟 | 狀態 |
|--------|------|
| 逐質數跡公式 Tr(e^{tΘ_p}) = log p·δ | ❌ 在 L²(R/log p Z) 上得到的是 δ 梳，不是單個 δ |
| 投影修正 | ⚠️ 可能但得非自伴算子 |
| 耦合使譜變成 ζ 零點 | ❌ 循環定義 |
| 非循環耦合的存在性 | ❌ 猜測，無根據 |

**結論**：T3 提供了分解框架但無法繞過解析延續障礙。
「加法分解 + 耦合」只是將原問題重新參數化。

# T4：Gauss-Manin 聯絡 + 參數族變形

## 核心想法

考慮 S_Z 上的一個參數族 s ∈ C，葉狀結構 F_s 隨 s 變化。
Gauss-Manin 聯絡 ∇_{GM} 在 H¹_F(S_Z × C) 上定義了平坦聯絡。
∇_{GM} 的單值群與 ζ(s) 的零點相關聯。

---

## 1. 結構

假設存在一個族：
```
π: X → C
```
其中 X_s = π⁻¹(s) ≅ S_Z 對所有 s ∈ C，但葉狀結構 F_s 在 X_s 上隨 s 變化。

Hodge 束 V = R¹π_*(Ω_{X/C})（相對 de Rham 上同調）是 C 上的向量叢，
帶有 Gauss-Manin 聯絡 ∇_{GM}。

### 1.1 與 ζ(s) 的關係

猜想：存在 C 上的全純線叢 L，使 V = L⊗V₀，且：
- ∇_{GM} 在 L 上的曲率 = (ζ'/ζ)(s)·ds ⊗ ds
- ∇_{GM} 的單值群 = ζ 零點的某種 monodromy 矩陣

### 1.2 函數域類比

對曲線族 C → P¹（Fermat 曲線族）：
- Gauss-Manin 聯絡的單值群 = 曲線的基本群表示
- Picard-Fuchs 方程 → 週期 → ζ 函數
- 單值的絕對值 → RH

---

## 2. 構造障礙

### 2.1 S_Z 不是代數簇

Deninger 的 S_Z = W_rat(Spec Z)(C) 是無窮維的，不是代數簇。
沒有「相對 de Rham 上同調」的標準理論。

### 2.2 族 π: X → C 不存在

S_Z 是 Spec Z 的「解析實現」。不存在一個非平凡族 S_Z → C。
Spec Z 沒有非平凡的自同構——它是「剛性的」。

### 2.3 無窮維 Hodge 理論不存在

即使我們接受 V = H¹_F(S_Z) 作為纖維，其上的 Gauss-Manin 聯絡
需要一個無窮維的變化 Hodge 結構理論——不存在。

---

## 3. 評估

| 子步驟 | 狀態 |
|--------|------|
| 族 X → C 的存在性 | ❌ Spec Z 剛性，無非平凡族 |
| 相對 de Rham 上同調 | ❌ 無窮維無定義 |
| Gauss-Manin 聯絡 | ❌ 不存在 |
| 單值 → RH | ❌ 不可操作 |

**結論**：T4 完全不可行。S_Z 沒有參數族可以變化。
